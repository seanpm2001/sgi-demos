GL3.2 Version 4.1 for AIX: Programming Concepts

List of GL Lighting Subroutines

getmmode
                          Returns the current matrix mode.
lmbind
                          Binds a new material, light source, or lighting model definition.
lmcolor
                          Respecifies the currently bound material properties.
lmdef
                          Defines a new material, light, or lighting model.
mmode
                          Sets the current matrix mode.
