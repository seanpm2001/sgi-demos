GL3.2 Version 4.1 for AIX: Programming Concepts

List of GL Depth-Cueing Subroutines

depthcue
                          Turns depth-cueing on and off.
getdcm
                          Indicates whether depth-cue mode is on or off.
lRGBrange
                          Sets the range of color indexes to use for depth-cueing in RGB mode.
lshaderange
                          Sets the range of color indexes to use for depth-cueing in color map mode.
