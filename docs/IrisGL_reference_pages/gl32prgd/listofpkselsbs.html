GL3.2 Version 4.1 for AIX: Programming Concepts

List of GL Picking and Selecting Subroutines

endpick
                          Turns off picking mode.
endselect
                          Turns off selecting mode.
gselect
                          Turns selecting mode on.
initnames
                          Initializes the name stack.
loadname
                          Loads a name on top of the name stack.
pick
                          Puts the system in picking mode.
picksize
                          Sets the dimensions of the picking region.
popname
                          Pops a name off the name stack.
pushname
                          Pushes a new name onto the name stack.
